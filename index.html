Connect Google Sheets — one-time setup (5 minutes)

Your app sends trade data to a Google Apps Script URL → the Script appends a row to your Sheet automatically. Completely free, no login required in the app.

1
Open Google Sheets, create a new spreadsheet. Name it PSX Trade Log.
2
Go to Extensions → Apps Script. Delete all default code and paste the script below exactly.
